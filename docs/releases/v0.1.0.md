# openclaw-sec v0.1.0

`openclaw-sec` is now publicly available as a local-first security audit tool for OpenClaw environments.

## Highlights

- Added the initial `openclaw-sec audit` CLI
- Added text, JSON, and Markdown report outputs
- Added detectors for:
  - OpenClaw config risks
  - plaintext secret exposure
  - filesystem permission issues
  - symlink escape risks
  - Git tracked-file secret scanning
  - Linux-first host and network checks
- Added masking rules so reports only keep redacted secret examples
- Added an OpenClaw skill wrapper
- Added English and Simplified Chinese documentation
- Added community files, security policy, and GitHub CI

## Why This Release Matters

V0.1.0 is intentionally narrow. The point is not to be a full security platform, but to provide a practical local audit that:

- runs with minimal setup
- produces high-signal findings
- degrades gracefully on unsupported checks
- avoids auto-remediation and network dependencies

## Included Report Formats

- terminal summary
- JSON report
- Markdown report

## Included Checks

- config existence and parseability
- heuristic OpenClaw config risk hints
- secrets in config, env files, backups, workspace docs, and logs
- Git current-tree secret scanning
- overly broad permissions on sensitive paths
- symlink escape detection
- listening port snapshots
- SSH hardening checks
- firewall and fail2ban detection
- permissive umask hints

## Notes

- This release does not perform automatic remediation.
- Git scanning covers the current tree, not full history.
- Some OpenClaw-specific findings are heuristic by design.
- Some host checks are best effort and may appear as `skipped` depending on platform or permissions.

## Getting Started

```bash
git clone https://github.com/haooyi/openclaw-sec.git
cd openclaw-sec
python3 -m pip install -e .
openclaw-sec audit
```

## Demo Fixture

This release also includes a generated demo fixture under `examples/redacted-lab/` so new users can try the tool against a safe synthetic environment without using real secrets.
